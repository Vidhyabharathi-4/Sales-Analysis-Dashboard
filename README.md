An Excel-based sales analysis dashboard providing insights into revenue, profit, sales trends, and regional performance.

## Business Objective
The objective of this project is to analyze sales data and present it in a clear and interactive dashboard format.  
This dashboard helps in understanding sales performance, identifying high and low performing regions and categories, and supporting data-driven business decisions.

---

## Key Performance Indicators (KPIs)
The dashboard tracks the following key metrics:
- Total Revenue  
- Total Profit  
- Profit Margin (%)  
- Total Units Sold  
- Sales by Category  
- Sales by Region  
- Sales by Payment Mode  
- Monthly and Annual Sales Trends  

---

## Insights
Key insights derived from the dashboard include:
- Office Supplies category shows strong sales performance  
- Some regions generate high revenue but have low profit margins  
- Sales trends vary across different months  
- A small number of categories contribute most of the overall profit

## Installation and Usage
1. Download the `sales_dashboard.xlsx` file
2. Open the file using Microsoft Excel
3. Use slicers to filter data by category, region, or time period
4. Analyze KPIs and charts to gain insights
